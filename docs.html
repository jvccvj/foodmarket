Chart Docs

📋 Product Requirements (PRD)

Goal: Public web visualization displaying seasonal fresh food prices as an interactive line chart with automatic weekly data refresh. No authentication required.

🎯 Key Requirements

  • Single HTML file, no backend, deployable via GitHub Pages
  • Line chart with 3 data series: Apples, Tomatoes, Milk
  • Data auto-refreshes weekly via GitHub Actions cron
  • Responsive design for mobile and desktop

📖 User Stories

  • US-001: View price chart — visitors see a clear line chart immediately on page load
  • US-002: Responsive on mobile — chart resizes on screens < 600px
  • US-003: Auto-refresh data — workflow updates CSV weekly (Mon 00:00 UTC) and redeploys
  • US-004: Readable metadata — footer shows units, date range, context

❌ Out of Scope (Non-Goals)

  • No user authentication or multi-user support
  • No custom data upload or chart configuration by users
  • No backend, database, or real-time API
  • No chart types beyond line charts
