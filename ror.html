Fresh Food Price Trends

Seasonal fruit, vegetable & dairy trends — Jan through Jun 2026

🥕 Prices in USD/kg  |  📊 6-month trend  |  🔄 Auto-refreshes every 60s

Tech Stack

Rails 7 Hotwire Turbo Stimulus Chart.js Tailwind CSS SQLite

Features

  • Auto-refresh every 60 seconds via Stimulus
  • Turbo Frames for seamless navigation
  • Loading skeletons while fetching
  • Responsive design with Tailwind
  • RESTful API endpoints
  • Active Record with SQLite

API Response

"prices": [ { "id": 1, "item": "Apples", "price": 3.50, "unit": "USD/kg", "month": "Jan", "recorded_at": "2026-01-15T00:00:00Z" }, ... ]

Built with Two Parallel Agents

This Rails app was constructed using OpenHands agents working in parallel:

Agent A — Backend Price model, API controller, migration, seeds
Agent B — Frontend Views, Stimulus controller, Chart.js integration

See rails-agent-log.md for full execution details.
